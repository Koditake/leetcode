# This is a LeetCode solution datadump.

The purpose is to store leetcode solutions to Github problems.

The format for those want to peruse this repository:

1. The answer keys are organized by their difficulties (Easy, Medium, Hard) and types (general programming & data).

2. You can search the problems using Leetcode's problem identification code (ie, [Merge Sorted Array](https://leetcode.com/problems/merge-sorted-array) can be search with number 88.)

2. Each question have one file, containing several approaches to solve the problem at hand, paired with explanantion.

3. All answers are written in C++/Java, with occasional Python/JS/Go.

4. Driver code and test cases are not included; though notable cases will be mentioned.
